# Не делать до версии 1.0

- Discord
- AI
- GitHub
- Jira
- Copilot
- MCP
- Голосовые сообщения (запись/отправка; инлайн-прослушивание — см. ADR-005,
  пока доступны как скачиваемый файл)
- Темы оформления
- Видеозвонки

## Отложено после 1.0

- Единое дерево нескольких провайдеров одновременно + `providerId` в моделях
  (ADR-009).
- WhatsApp: медиа, поиск, профили, mute (ADR-010).
  Пагинация истории (ADR-011) и read-receipts (ADR-012) — сделаны в v1.1.
