# Yapper

> Unified Developer Messenger for VS Code

## Идея проекта

Yapper — это расширение для Visual Studio Code, которое объединяет рабочие мессенджеры в едином интерфейсе и использует возможности IDE для повышения продуктивности разработчика.

Первая версия проекта ориентирована на Telegram. В дальнейшем архитектура должна позволять подключать другие мессенджеры (WhatsApp, Slack, Discord, Teams и т.д.).

Главная цель проекта — не заменить официальные клиенты, а предоставить разработчику инструменты, которых нет в обычных мессенджерах.

---

# Проблема

Во время разработки приходится постоянно переключаться между:

* VS Code
* Telegram
* WhatsApp
* Slack
* Discord
* Jira
* Browser

Это приводит к постоянной потере контекста.

---

# Цель MVP

Создать Telegram-клиент, полностью интегрированный в VS Code.

Пользователь должен иметь возможность:

* авторизоваться;
* просматривать список чатов;
* читать сообщения;
* отправлять сообщения;
* получать новые сообщения в реальном времени.

После MVP расширением уже можно пользоваться ежедневно.

---

# Принципы разработки

* MVP прежде всего.
* Каждый этап должен давать рабочий результат.
* Не добавлять функциональность без реальной необходимости.
* Делать модульную архитектуру.
* Не привязываться к одному мессенджеру.
* Максимально использовать возможности VS Code вместо копирования интерфейсов существующих клиентов.

---

# Критерии готовности MVP

* Пользователь успешно авторизуется в Telegram.
* Отображается список чатов.
* Можно открыть чат.
* Загружается история сообщений.
* Сообщения обновляются в реальном времени.
* Можно отправить сообщение.
* Сессия сохраняется между перезапусками VS Code.
