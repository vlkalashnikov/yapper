## v0.7

- Localization / i18n (ADR-008)

  - [x] инфраструктура: `package.nls*.json` + `l10n/` (`vscode.l10n`);
  - [x] package.json → `%ключи%` (команды, конфиг, welcome, view);
  - [x] extension host → `vscode.l10n.t` (тосты, диалоги, ошибки, медиа-подписи);
  - [x] webview → словарь `window.L10N`, инъекция из host;
  - [x] EN базовый, RU — перевод; язык следует настройке VS Code;
  - [ ] позже: другие языки (добавить `*.<lang>.json`).

## v0.8

- Поиск / Search

  - [x] Поиск по чатам — QuickPick (кнопка в хедере списка + палитра + Cmd/Ctrl+Alt+C)
  - [x] Поиск сообщений в открытом чате — 🔍 в хедере диалога, QuickPick + переход
  - [x] Глобальный поиск по всем чатам — `messages.searchGlobal`, QuickPick
        (кнопка в хедере списка + палитра + Cmd/Ctrl+Alt+G)

- Профиль

  - [x] Карточка профиля чата/собеседника — оверлей по клику на шапку диалога

        общие группы, ссылка-приглашение с копированием); действия: mute/unmute,
        поиск по чату; вкладки Медиа/Файлы (сетка превью + список файлов)
