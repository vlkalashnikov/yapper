## v1.1

- WhatsApp (ADR-009, ADR-010)

  - [x] Провайдер через Baileys, text-only — вход по QR, список чатов,
        история, отправка, realtime;
  - [x] Мульти-провайдер: переключение активного, тег BETA в UI;
  - [x] Пагинация истории — догрузка старых сообщений on-demand через Baileys
        `fetchMessageHistory` (ADR-011);
  - [x] Read-receipts — галочки ✓/✓✓ для исходящих в личке через Baileys
        `messages.update` (ADR-012);
  - [x] Mute — read-only: уважаем мьюты WhatsApp (тихие чаты не пикают
        тостами); тумблер — позже с карточкой профиля (ADR-013);
  - [x] Editor-парити — `sendCode`/`sendFile`: Share Selection/Diff/File
        работают на WhatsApp (код — triple-backtick, файл — документ) (ADR-014);
  - [x] Медиа (входящие) — превью из `jpegThumbnail`, скачивание/открытие
        через `downloadMediaMessage`; «[media]» заменены реальными
        картинками/файлами; медиа-метки в списке чатов (ADR-016);
  - [ ] Аватары, shared-media, поиск, профили (см. NOT_NOW / ADR-010) —
        **в работе**.
