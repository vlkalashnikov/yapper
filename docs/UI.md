# UI

Стиль — как в расширении Claude Code (см. ADR-002).

## Расположение

Activity Bar → Yapper

Сайдбар Yapper:

- Chats (TreeView) — список чатов
- Conversation (Webview) — открытый диалог

## Chats (TreeView)

- иконка чата
- название
- превью последнего сообщения
- счётчик непрочитанных

## Conversation (Webview)

Стиль Claude Code:

- цвета только из темы VS Code
- вертикальный список сообщений, без пузырей
- автор + текст, свои сообщения помечены
- поле ввода снизу
- позже: markdown, слэш-команды, @-упоминания файлов
